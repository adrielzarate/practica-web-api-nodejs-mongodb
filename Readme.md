nodepop
=======

## Instrucciones de instalacion

1. Instalamos las dependencias
```
    npm install
```
2. Si se quiere empezar al aplicacion con datos precargados en la base de datos
```
    npm run install_db
```
3. lanzar aplicacion en modo debug
```
    npm run dev
```
4. lanzar applicacion
```
    npm start
```
